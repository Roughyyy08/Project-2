# Personal Finance & Expense Analytics App

A React-based personal finance tracker that helps users record income/expenses, analyze spending patterns, manage budgets, and visualize financial trends.

## Features
- Add, edit, and delete transactions
- Search, filter, and sort transaction history
- Track monthly budgets with summary cards
- Analytics dashboard with charts and recurring expense highlights
- React Router navigation and responsive UI
- Local storage persistence

## Scripts
- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Tech stack
- React
- Vite
- React Router DOM
- React Hook Form + Yup
- Recharts
- Axios
- React Toastify
- Framer Motion
- Date-fns
- UUID
